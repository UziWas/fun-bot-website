Fun Bot Commands

  • .giveadmin

    Gives you the "Admin" role with all permissions to manage the server. Perfect for server admins!

  • .dmall <message>

    Send a custom direct message to all members in the server, excluding bots. Use this to keep everyone in the loop!

  • .createchannel <channel_name>

    Create a new private text channel that only you can access. Ideal for private discussions or events.

  • .createvcchannel <vc_channel_name>

    Create a new private voice channel that only you can access. Host private voice chats with friends or colleagues.
