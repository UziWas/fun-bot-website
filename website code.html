Fun Bot - The Ultimate Discord Bot

Available Commands

.help

Displays a list of all commands.

.dmall <message>

Send a message to all members (excluding bots) in the server.

.createchannel <channel_name>

Create a private text channel just for you!

.createvcchannel <vc_channel_name>

Create a private voice channel just for you!

.giveadmin

Gives the "Admin" role with all permissions to the user.
